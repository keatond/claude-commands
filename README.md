# Claude Commands

Custom Claude Code slash commands.

## Installation

Copy any `.md` file from this repo into `~/.claude/commands/`. The command becomes available immediately as `/<filename>` in Claude Code.

## Commands

| Command | Description |
|---------|-------------|
| `/project-plan` | Structured project planning — grill-me style discovery interview, team design, ticket breakdown with acceptance criteria, and QA monitor protocol |
